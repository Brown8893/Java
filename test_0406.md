```
Armstrong數
問題描述：
所謂 " Armstrong數 " 是指一個三位數的整數，其各位數字之立方和等於該數本身。例如： 153 是一個 Armstrong數，因為 153 =1 3 + 53 + 33 。試撰寫一程式，判斷是否為 Armstrong數。

輸入說明：
輸入一個三位數正整數。

輸出說明：
是 阿姆斯壯數輸出 Yes ，不是 阿姆斯壯數輸出 No 的訊息。
```

```
import java.util.Scanner;    
import java.lang.Math;  
public class Main {    
            
    /**  
     * Creates a new instance of <code>Main</code>.  
     */    
    public Main() {    
    }    
        
    /**  
     * @param args the command line arguments  
     */    
    public static void main(String[] args) {   
        int a,b,c,d;  
        double s;    
            
        Scanner sc= new Scanner(System.in);    
            
         while(sc.hasNext()){               
            a=sc.nextInt();    
                  
            if(a>999 || a<100){  
                continue;  
            }  
              
            b=a%10;  
            c=(a%100)/10;  
            d=a/100;  
            s=Math.pow(b,3)+Math.pow(c,3)+Math.pow(d,3);  
              
            if((int)s==a){  
                System.out.println("Yes");  
            }else{  
                System.out.println("No");  
            }  
  
              
          }    
    
            
    }    
         
}
```
